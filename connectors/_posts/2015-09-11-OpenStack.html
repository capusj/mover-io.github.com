Mover and {{ page.html_title }}

OpenStack is a cloud computing project that aims to provide an open source cloud computing platform. The OpenStack Object Storage (Swift) is a scalable redundant system that ensures safe and secure online cloud storage. Mover is able to connect with the OpenStack infrastructure and API while securely transferring files in and out of any service that use this open source software.

With Mover it is simple and fast to move files in and out of the systems that use the OpenStack cloud structure. It is always important to ensure that your files are in more than one location. This Connector is bi-directionally compatible with all of our other Connectors, as well it can utilize the full range of tools offered by Mover. Transferring files on a regular schedule.

  • Transferring to or from OpenStack clouds from a separate cloud storage service.
  • Sharing huge amounts of data on a different service like Box.
  • Simply creating a backup of your working data to another system like Dropbox.
  • Creating a schedule to manage file transfers and backups with OpenStack and other Connectors.

File security is taken very seriously here at Mover and we protect every file you move in or out of OpenStack. All transfers are encrypted end-to-end and we are committed to continuously reviewing and improving your file security. Our Security Overview is a great read, and we encourage all our uses to take a moment to read through it.

With the Mover OpenStack Connector, managing your files in OpenStack services has never been easier.
